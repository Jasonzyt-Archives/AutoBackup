# AutoBackup
## A Easy AutoBackup Tool
### 简体中文
如何使用自动备份?
+ 修改 config.ini
+ 运行 AuoBackup.exe
备注:本软件依赖于Bandzip的控制台应用bz.exe(需要ark.xxx.xxx.dll动态链接库),用于压缩文件(夹) 不过您也可以自行修改代码(方法如下)换成其他软件进行压缩(如7z.exe)
### English
How to use Autobackup?
+ Edit config.ini
+ Run AutoBackup.exe
NOTE:This software depends on the console application of Bandzip(bz.exe,ark.xxx.xxx.dll) to compress file(clips).but you can change the code to other software for compression(such as 7z.exe)
